## 🎯 Project Overview

SignBridge is an interactive web application designed to make learning **American Sign Language (ASL)** engaging, accessible, and effective. It bridges the communication gap between hearing and deaf communities by providing structured lessons, real-time practice with camera feedback, quizzes, and gamified progress tracking.

### Key Features

- **Structured Lessons**
  - ASL Alphabet (A–Z)
  - Numbers 1–10
  - Common Phrases

- **Practice Studio**
  - Real-time camera mode using **MediaPipe Hands**
  - Image-only practice mode (for Numbers & Phrases)
  - Word Challenge mode (spell words letter by letter)

- **Interactive Quiz System**
  - Lesson-based quizzes with scoring
  - Smart lesson unlocking (70%+ to pass)

- **Gamification**
  - XP points, levels, daily streaks
  - Mastery badges and achievements

- **Progress Tracking**
  - User profile with detailed stats
  - Saved progress in database

## 🛠️ Technology Stack

### Frontend
- HTML5, CSS3, JavaScript (Vanilla)
- **MediaPipe Hands** – Real-time hand landmark detection
- Responsive modern dark UI

### Backend
- PHP
- MySQL (User data, progress, quiz results)

### External Resources
- High-quality ASL images from [Lifeprint.com](https://www.lifeprint.com)
